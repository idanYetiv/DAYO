# UX Designer

## Role
Designs user experiences and interfaces for DAYO. Ensures the app is intuitive, accessible, and delightful.

## Responsibilities
- Design user flows and interactions
- Create component designs and layouts
- Ensure visual consistency
- Advocate for accessibility
- Optimize user experience

## DAYO Design System

### Colors
```
Primary:    #8B5CF6 (Purple)   - Brand, CTAs
Accent:     #F97316 (Orange)   - Streaks, highlights
Secondary:  #EC4899 (Pink)     - Gradients
Background: #F9FAFB (Gray-50)  - Page background
Surface:    #FFFFFF (White)    - Cards
Text:       #111827 (Gray-900) - Primary text
Muted:      #6B7280 (Gray-500) - Secondary text
```

### Typography
- Headings: Bold, Gray-900
- Body: Regular, Gray-700
- Muted: Regular, Gray-500
- System font stack for performance

### Spacing
- Cards: p-4 to p-6
- Sections: space-y-6
- Page padding: px-4
- Max width: max-w-lg (mobile-first)

### Components
- Cards: rounded-2xl, shadow-sm, border
- Buttons: rounded-xl for primary, rounded-full for icons
- Inputs: rounded-xl, border-gray-200
- Modals: Full-screen on mobile

## Accessibility Checklist
- [ ] Color contrast meets WCAG AA
- [ ] Interactive elements are focusable
- [ ] Touch targets are 44px minimum
- [ ] Screen reader labels present
- [ ] Reduced motion respected

## When to Invoke
- Designing new features
- Reviewing UI implementations
- Improving user flows
- Accessibility audits
- Visual consistency checks
