## Tagline

## Core Features (All DB-Connected)

| Feature | Description | Status |
|---------|-------------|--------|
| **Authentication** | Email/password via Supabase Auth | ✅ |
| **Daily Planner** | Create, complete, delete tasks per day | ✅ |
| **Diary** | Mood tracking + free-form journaling | ✅ |
| **Photo Upload** | Up to 5 photos per diary entry | ✅ |
| **Gratitude** | 3 daily gratitude items | ✅ |
| **Highlights** | 5 daily highlights with emojis | ✅ |
| **Calendar** | Navigate to any date, view history | ✅ |
| **Goals** | Yearly/Monthly/Weekly with milestones | ✅ |
| **Habits** | Daily/weekly tracking with streaks | ✅ |
| **Streak System** | Consecutive day tracking | ✅ |
| **AI Assistant** | Chat interface (mock mode ready for OpenAI) | ✅ |
| **Instagram Export** | 3 templates, 3 styles, Story/Post formats | ✅ |
| **Settings** | Profile, preferences, theme, password, export, delete | ✅ |
| **Kids/Adults Mode** | Two distinct experiences with different themes | ✅ |
| **Onboarding** | Profile type selection for new users | ✅ |
| **Landing Page** | Public marketing page with mode preview | ✅ |
| **iOS App** | Capacitor-based native iOS app | ✅ |

## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React 19 + TypeScript |
| Styling | Tailwind CSS |
| Build | Vite 7 |
| Routing | React Router 7 |
| Server State | React Query 5 |
| Client State | Zustand |
| Backend | Supabase (Auth, Database, Storage) |
| Database | PostgreSQL with RLS |
| Hosting | Vercel |
| Testing | Vitest (79 tests) + Playwright E2E |

## Value Proposition

| Value | How DAYO Delivers |
|-------|-------------------|
| **Productivity** | Daily tasks + habits + goals in one place |
| **Mental Wellness** | Mood tracking, gratitude journaling, reflection |
| **Self-Improvement** | Streak gamification, milestone tracking |
| **Social Sharing** | Instagram export for accountability |
| **Privacy** | All data user-owned, export/delete anytime |
| **AI-Ready** | Assistant for summaries and insights |
